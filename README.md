# IterationRP Patches

## Features

- OpenDRT display rendering transform (tonemapping) by [Jed Smith](https://github.com/jedypod/open-display-transform) for polished and rich color rendering.
- More in the future.

## How to apply

```bash
# For Linux and MacOS
patch -p1 -d <path_to_folder> < itrp.patch

# On Windows with Git Bash
git apply -p1 --directory=<path_to_folder> itrp.patch
```
